# Postman Course Collections

This repository contains Postman collections created during API testing practice and courses. It includes example requests, scripts, and environments for learning and testing various APIs.

## Collections Included

1. **Simple Grocery Store API SCRIPTS**  
   - API requests and scripts for testing a grocery store API.

2. **Simple Grocery Store API**  
   - Basic collection for testing endpoints of the grocery store API.

3. **Trello API**  
   - Collection for testing Trello API endpoints.

4. **Valentine's Book List**  
   - Example collection demonstrating requests to a book list API.
